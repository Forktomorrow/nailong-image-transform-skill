# 示例

## 《星月夜》式画面

元素分析：

- 月亮与星星是主要视觉锚点。
- 天空旋涡、柏树、村庄和山脉用于识别原构图，不应替换。
- 每个天体的大小、位置和发光关系必须保留。

提示词：

```text
编辑输入图像，把天空中的月亮和主要星星分别替换成奶龙。每个奶龙继承对应天体的中心位置、大小和发光强度：大月亮变成蜷缩坐姿的发光奶龙，小星星变成较小的漂浮奶龙头部或全身。保持天空旋涡、柏树、村庄、山脉、画幅和所有空间关系不变。奶龙必须使用与原画一致的厚重油彩、短促旋转笔触和蓝黄配色，看起来像原画的一部分，而不是贴上去的卡通贴纸。不要增加天体数量，不要修改地景，不要添加文字或水印。
```

示例成图位于 `assets/examples/starry-night-nailong-example.jpg`。

## 双人反应表情包

元素分析：两个人物是笑点载体，字幕与背景不替换。

```text
只把画面中的两个人物替换成两只奶龙。左侧奶龙复现左侧人物的身体倾斜、手势和表情；右侧奶龙复现右侧人物的朝向、视线和动作。保持两者的相对距离、遮挡、背景、镜头、光照和所有字幕逐字不变。不要增加第三只奶龙。
```

## 俯视旋转楼梯

元素分析：

- 灰色踏步形成全图最强的圆形旋涡意象。
- 白色栏杆、中心洞口和俯视透视用于识别建筑空间，必须保留。
- 应用一只完整蜷缩奶龙继承旋涡轮廓，不能拉成长蛇或普通黄色楼梯。

```text
只把灰色旋转楼梯结构替换成一只从正上方看、身体蜷成约一圈半的巨大奶龙。奶龙的圆头靠近内圈并抬眼看向镜头，肥圆身体和浅色肚皮沿楼梯原来的旋涡宽度盘绕，短手短脚收在身体两侧，小尾巴在外圈收尖。保持白色金属栏杆、中心黑色洞口、混凝土边缘、俯视镜头、冷蓝光线、积雪质感和所有建筑关系不变。必须是一只完整肥圆的奶龙，不是蛇、虫、管道、黄色通道或多只奶龙队列。
```

## 单一物体梗图

元素分析：只替换被指代的物体，避免把手、桌面或文字一起重绘。

```text
只把桌面中央的圆形闹钟替换成一只蜷缩成闹钟轮廓的奶龙，保持原闹钟的尺寸、中心位置、倾角、投影和被手触碰的关系。保留手、桌面、背景和全部文字不变。奶龙需要清楚可辨，但仍承担原闹钟的视觉位置和笑点功能。
```
