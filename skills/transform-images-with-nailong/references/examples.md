# 示例

## 《星月夜》式画面

元素分析：

- 月亮与星星是主要视觉锚点。
- 天空旋涡、柏树、村庄和山脉用于识别原构图，不应替换。
- 每个天体的大小、位置和发光关系必须保留。

提示词：

```text
编辑输入图像，把天空中的月亮和主要星星分别替换成奶龙。每个奶龙继承对应天体的中心位置、大小和发光强度：大月亮变成蜷缩坐姿的发光奶龙，小星星变成较小的漂浮奶龙头部或全身。保持天空旋涡、柏树、村庄、山脉、画幅和所有空间关系不变。奶龙必须使用与原画一致的厚重油彩、短促旋转笔触和蓝黄配色，看起来像原画的一部分，而不是贴上去的卡通贴纸。不要增加天体数量，不要修改地景，不要添加文字或水印。
```

示例成图位于 `assets/examples/starry-night-nailong-example.jpg`。

## 双人反应表情包

元素分析：两个人物是笑点载体，字幕与背景不替换。

```text
只把画面中的两个人物替换成两只奶龙。左侧奶龙复现左侧人物的身体倾斜、手势和表情；右侧奶龙复现右侧人物的朝向、视线和动作。保持两者的相对距离、遮挡、背景、镜头、光照和所有字幕逐字不变。不要增加第三只奶龙。
```

## 俯视旋转楼梯

元素分析：

- 灰色踏步形成全图最强的圆形旋涡意象。
- 白色栏杆、中心洞口和俯视透视用于识别建筑空间，必须保留。
- 楼梯属于长路径对象，不是月亮式紧凑圆团；应用一只长条奶龙沿整条螺旋连续盘绕。

```text
只把灰色旋转楼梯结构替换成一只从正上方看、身体沿整条楼梯中心线连续盘绕的长条奶龙。圆钝大头放在螺旋内端并与身体连续相接，抬起高位大眼看向镜头；根部厚的尾巴位于最外端并自然收束。身体带宽等于原踏步带宽，浅色肚皮沿螺旋内侧形成连续长带，短粗手脚贴在主要转折处，身体表面保留踏步节奏形成的细微横向褶皱。保持白色金属栏杆、中心黑色洞口、混凝土边缘、俯视镜头、冷蓝光线、积雪质感和所有建筑关系不变。必须是一只有明确头、粗尾、连续肚皮和短粗肢体的长条盘旋奶龙，不是多只奶龙队列，也不是无结构的黄色管道。
```

## 东方绘画经验

- 山水：让奶龙和山体形成双重识别。树木、皴法、瀑布与岩石纹理可以生长在奶龙背上，远看仍维持山势。
- 青绿山水：允许奶龙变为石青、石绿或金色。角色识别依赖比例、肚皮分区与五官，不要求固定亮黄色。
- 奔马或运动主体：继承速度、重心、腾空关系和笔触方向；动作功能比原物种轮廓重要。
- 飞天或舞蹈主体：用飘带、衣纹和云气延续运动轨迹，但保持奶龙主体轮廓清楚。

## 单一物体梗图

元素分析：只替换被指代的物体，避免把手、桌面或文字一起重绘。

```text
只把桌面中央的圆形闹钟替换成一只蜷缩成闹钟轮廓的奶龙，保持原闹钟的尺寸、中心位置、倾角、投影和被手触碰的关系。保留手、桌面、背景和全部文字不变。奶龙需要清楚可辨，但仍承担原闹钟的视觉位置和笑点功能。
```
