## 为什么不能直接给未测案例打分

扩展集的目标是防止过拟合。若把模型已经生成的奶龙图、临时合成背景或没有 mask 的图片直接拿来评分，评测会同时改变输入、目标和标准，结果没有可比性。下一轮必须为每个案例固定：原图、目标 mask、遮挡层、输出尺寸、随机种子和盲评表。

## 当前优化结论

纯算法当前真实成绩是 `67/100`，仍未达到可用线。下一步应先实现自适应拓扑生成器，再补齐扩展集的原图/mask 对；不能通过增加样例数量或调整权重虚增分数。
