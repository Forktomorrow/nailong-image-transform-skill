# 目标形状到奶龙姿态语法

固定站立奶龙不是通用替换器。下表是姿态库的示例标签和验收案例，不是场景硬编码；实际运行必须由 [adaptive-topology-generator.md](adaptive-topology-generator.md) 根据 mask 特征检索和连续变形。

| 原物体几何/语义 | 奶龙姿态语法 | 关键约束 |
| --- | --- | --- |
| 圆形星体、月亮、新月 | 蜷缩/抱月奶龙 | 身体沿圆弧或月牙内侧弯曲，头尾相向，肚皮朝内 |
| 长路径、楼梯、河流、丝带 | 螺旋/盘绕奶龙 | 身体中心线就是路径，头尾落在端点，宽度随原路径变化 |
| 山体、云团、丘陵 | 山脊奶龙 | 背部轮廓承担山势，肚皮作为内侧明暗带，远看仍是山 |
| 奔马、人物动作、飞天 | 动势奶龙 | 先提取重心、速度方向和骨架，再继承受力关系 |
| 手、器物、局部符号 | 局部器官/抱持奶龙 | 只生成承担该功能的局部姿态，不能塞入整只站立奶龙 |
| 重复星点、树叶、印章 | 小型表情/姿态阵列 | 数量、节奏、疏密继承原图 |

## 新月生成规则

1. 对新月 mask 提取两条边界和主弧线。
2. 用内侧弧线作为奶龙腹部中心线，按弧长生成连续身体。
3. 头部放在弧线较宽的一端，尾巴收进另一端；头尾相向形成“抱月”。
4. 四肢只在转折处出现，避免破坏月牙轮廓。
5. 从原月亮周围的环带采样颜色、笔触方向和辉光；奶龙拓扑只改变轮廓和面部锚点。

## 风格场约束

姿态生成后不能直接填纯黄色。对周围环带估计颜色、亮度、纹理尺度和笔触方向，把统计投影到奶龙身体；面部、肚皮和尾巴只作为结构锚点叠加。油画使用方向性短笔触，水墨使用墨色浓度和干湿边缘，壁画使用平涂色块与轮廓线，照片使用局部景深和噪声。
