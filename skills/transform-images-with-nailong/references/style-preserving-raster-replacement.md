# 保持原画风格的像素级替换

前一版预览失败的原因很明确：它使用了透明奶龙 PNG，执行的是 alpha compositing（贴图），而不是对原画中的目标物体做重建。合格的“奶龙化”应当只改变目标物体的几何身份，尽量复用该物体原本的笔触、颗粒、光照和颜色分布。

## 正确的处理顺序

```text
原图
  └─ 目标物体 mask
      ├─ 物体内部的笔触/纹理/光照统计
      ├─ 轮廓、骨架、宽度场、遮挡关系
      └─ 奶龙拓扑模板
            ↓
      形状重建 + 局部纹理重采样 + 梯度域融合
            ↓
      原画风格不变、目标物体变成奶龙
```

## 1. 只处理目标物体

优先使用人工 mask；没有 mask 时使用边缘、颜色聚类、连通域、GrabCut/分水岭和形态学开闭运算得到候选区域。对候选区域做轻度高斯平滑和轮廓简化，避免把油画笔触误识别成身体边界。背景和未选区域必须作为冻结层，不能重新绘制。

## 2. 从原物体提取可继承信息

在 mask 内计算：

- 轮廓 `C`、骨架 `S`、距离变换 `D` 和主方向；
- 亮度/色相/饱和度的局部统计；
- 高频笔触方向场 `G`（可由 Sobel/Scharr 梯度估计）；
- 前景与背景的深度边缘和遮挡顺序。

这些信息不是用来生成一个新背景，而是作为奶龙表面的“原画材质约束”。例如《星空》的奶龙身体应沿原星体的旋涡笔触着色，而不是保留一张摄影风格 PNG。

## 3. 用奶龙拓扑模板重建形状

奶龙模板用矢量路径表达，不使用现成位图。头身是连续闭合轮廓，腹部是内侧闭合区域，四肢和尾巴是有宽度的附属路径，眼鼻口是固定锚点。将模板通过弧长参数映射到 `S`，通过 `D` 调整局部宽度，通过曲率选择站立、侧卧或蜷缩姿态。这样得到的是“奶龙形状的 mask”，不是奶龙图片。

## 4. 把原画材质转移到新形状

对新 mask 内的每个像素，按其到模板中心线的弧长和法向坐标，在原物体内部采样对应的亮度、色相和笔触方向；再用局部均值/方差匹配恢复色彩范围。高频梯度沿 `G` 方向重新投影，低频光照沿原物体的明暗场继承。这样只改变轮廓和结构，笔触仍然来自原画。

## 5. 边缘融合而不是贴图

使用窄边带的羽化、梯度域 Poisson 融合或多频带融合处理新旧边界。融合只发生在目标 mask 的边缘，不允许覆盖背景。对于被目标物体遮挡的区域，用其邻域纹理做 patch-based inpainting；不能把整张图交给一个新素材覆盖。

## 6. 验收标准

一张图只有同时满足以下条件才算成功：

1. 未选区域的像素和构图没有明显漂移；
2. 目标区域轮廓是奶龙，而不是黄色椭圆或贴图边框；
3. 目标区域内部的笔触方向、噪声、光照与原画一致；
4. 目标物体的原功能和位置仍然可读；
5. 放大观察看不到透明 PNG 的白边、摄影质感或不匹配的景深。

前一版 `procedural/*-fit` 仅作为速度和布局演示，不满足以上验收标准，不能当作最终效果展示。
