# 不依赖生图模型的拟合算法

卷积和池化本身不是生成器：它们可以提取边缘、纹理和语义特征，但不能凭空重建奶龙的头、肚皮、四肢和尾巴。更稳的无模型方案是“轻量视觉分析 + 奶龙资产几何拟合 + 遮罩合成”。

## 核心思想：把目标区域变成一个可优化的形状场

不要直接把目标区域涂黄。设目标二值区域为 `M(x,y)`，先计算它的距离变换 `D(x,y)`、骨架 `S` 和边界曲率 `κ`。奶龙不是一个椭圆，而是一个带拓扑约束的参数化模板 `N(θ)`：连续头身、腹部内侧带、四肢锚点、尾巴锚点、眼鼻口锚点。对每个候选位置和尺度，优化下面的代价：

```text
E = λ1 * uncovered(M, N)
  + λ2 * outside(N, M)
  + λ3 * curvature_mismatch(boundary(M), boundary(N))
  + λ4 * topology_penalty(N)
  + λ5 * overlap_with_previous(N)
```

其中 `topology_penalty` 会惩罚断头、无肚皮、尾巴过细、四肢消失和眼鼻口错位；因此即使目标轮廓很怪，也不能退化为黄色胶囊。先求一只大奶龙，再对残余区域重复求解中奶龙和小奶龙：

```text
R0 = M
for scale in [large, medium, small]:
    candidates = sample_pose_scale_position(R_previous, S, κ, D)
    best = argmin(E(candidate))
    if coverage(best, R_previous) >= threshold:
        place(best)
        R_previous = R_previous - visible_mask(best)
```

这就是“分形”在工程上的可控版本：不是把同一张图无限缩小，而是对同一套奶龙拓扑模板，在不同尺度、姿态和曲率半径下递归求解。大奶龙承担主体语义，中奶龙覆盖分叉和缺口，小奶龙填窄缝；每一级都重新估计姿态和遮挡。

## 推荐流水线

1. **对象提取**：优先使用用户提供的 mask；否则用边缘、颜色聚类、显著性和连通域得到候选对象。需要语义识别时可选本地视觉模型，但不把它和生成模型混为一谈。
2. **形状表示**：对候选区域计算轮廓、骨架/中心线、距离变换和主方向。长路径用骨架，块状对象用轮廓，窄缝与凹角用距离变换的局部极小值。轮廓先用高斯平滑和形态学开闭运算去掉毛刺，再用 B-spline 或分段 Bézier 拟合，避免锯齿驱动出畸形奶龙。
3. **模板变形**：将奶龙模板转成局部坐标，沿骨架的切线/法线建立平行移动标架；长度由弧长映射，宽度由 `D` 的中位数映射，曲率大的位置自动切换蜷缩或侧卧姿态。头、肚皮、四肢、尾巴和面部锚点随同一变换移动，不能分别贴图。
4. **多尺度拟合**：先放置一只大奶龙覆盖主体，再在残余区域放置中奶龙，最后用小奶龙填补凹角和窄缝。每只资产都必须保留头身连续、浅色大肚皮、短粗四肢、粗短尾和高位高光眼；禁止用黄色椭圆或色块代替。
5. **姿态与布局**：沿中心线估计切线方向，旋转资产；用宽度场控制缩放；在急转弯处使用侧卧/蜷缩姿态。头尾落在语义端点，肚皮朝向路径内侧，避免“奶龙素材堆在一起”。
6. **遮罩与合成**：先做前景/背景分层，再按深度顺序合成。边缘用羽化和 Poisson/多频带融合；内部纹理使用原区域的亮度、噪声和笔触统计重新着色，保留原图媒介，不凭空生成新背景。
7. **验收**：检查覆盖率、头尾可见性、肚皮连续性、姿态方向、原构图和文字。若局部不合格，只重算该对象，不重做整张图。

## 适用边界

该路线非常适合前端控件、卡片、徽标、路径、旋转楼梯和规则插画；对复杂名画、真实人物、自然遮挡和全局画风重绘，仍可用同一套算法先得到结构化初稿，再选择性让模型做边缘和材质修复。也就是说，模型不是核心几何器，而是可选的最后一层媒介融合器。

## 向高质量示例靠近的优化规则

1. **先锁定对象，再锁定风格**：先得到稳定的目标 mask 和遮挡图，禁止用全图滤镜或全图重绘破坏背景。
2. **由位图贴合改为矢量拓扑渲染**：奶龙只以路径和锚点存在；身体、肚皮、四肢、尾巴和面部必须在同一坐标变换中生成。
3. **原画材质优先**：对奶龙新 mask 采样原物体的低频光照和高频笔触，不使用摄影奶龙 PNG 的原始纹理。
4. **边缘分两次融合**：先用窄边带匹配颜色和梯度，再做局部 Poisson/多频带融合；背景区域不可参与优化。
5. **控制奶龙数量**：数量由残余面积和曲率决定，而不是强度滑杆直接增加贴图。目标是“一只大奶龙解决主要语义，少量中小奶龙解决几何缺口”。
6. **失败优先级**：风格/融合低于 50 分时，不继续调色或增加角色，先修正 mask、拓扑渲染和纹理重采样。

使用 `references/benchmark.md` 评估，70 分以上才算可用，50–69 分仅算布局演示。
