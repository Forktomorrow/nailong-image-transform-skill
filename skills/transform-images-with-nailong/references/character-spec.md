# 奶龙视觉规格

## 必须保留的识别特征

- 黄色、圆润、矮胖的幼龙/恐龙式吉祥物轮廓。
- 梨形或豆形身体，头身连接自然，腹部饱满。
- 奶油白或浅黄色椭圆肚皮。
- 小而简洁的黑色眼睛，极简鼻口，神态憨萌、迟钝或无辜。
- 短手、短腿、小脚和短尾巴；动作笨拙但清楚。
- 表面柔软顺滑，可随输入图媒介变为油画笔触、像素块、铅笔线、黏土或摄影材质。

## 允许的适配

- 为匹配月亮、太阳、气泡或圆形图标，可蜷缩身体或使用圆润头部轮廓。
- 为匹配楼梯、漩涡、圆环等强轮廓对象，可让一只完整奶龙从俯视角蜷成一至两圈；身体保持肥圆，头、肚皮、短肢与尾巴必须仍可辨认。
- 为匹配人物，可让奶龙复现原人物的姿势、视线、手势和情绪。
- 为匹配长条、弧形或倾斜物体，可用身体弯曲、尾巴和四肢形成相近轮廓。
- 黑白图像中允许去除黄色，但必须保留体型、肚皮分区和五官结构。

## 禁止漂移

- 不变成普通黄色鸭子、鸡、熊、恐龙玩具或写实爬行动物。
- 不为追随旋涡轮廓而拉成长蛇、毛毛虫、管道或没有完整身体结构的黄色带状物。
- 不增加复杂鳞片、尖牙、巨大翅膀、长角、肌肉或凶猛表情。
- 不因追求写实而丢失极简五官和圆润比例。
- 不在没有依据时添加服装、帽子、武器、文字或品牌标识。

## 一行描述

`奶龙：黄色圆润矮胖的幼龙吉祥物，梨形身体、浅色椭圆肚皮、极简黑色小眼睛和鼻口、短手短腿与小尾巴，神态憨萌无辜。`
