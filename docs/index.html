03 / THE SKILL

奶龙化，不等于
把所有东西都变黄。

  1. 01

    找出重要元素

    优先识别主角、最大物体、核心道具、月亮、太阳和重复视觉符号。

  2. 02

    建立替换映射

    让奶龙继承原元素的位置、尺度、朝向、动作、遮挡与画面功能。

  3. 03

    锁定其余内容

    文字、背景、构图、透视和未指定元素默认保持，减少模型整图重画。

  4. 04

    继承原图媒介

    油画保留笔触，照片匹配光影，像素图保持网格，漫画延续线条与网点。

QUICK PROMPT

把这张图中决定主题和笑点的重要元素逐一替换成奶龙。每只奶龙继承原元素的位置、大小、姿态、透视和画面功能；严格保留原构图、背景、文字和画风。不要把整张图染黄，不要随机增加角色。
